## Run Locally
```bash
git clone git@github.com:liubko/elm-snake.git
cd elm-snake
elm-reactor
```

And then open [http://localhost:8000/SnakeGame.elm?debug](http://localhost:8000/SnakeGame.elm?debug) to see it in action!
